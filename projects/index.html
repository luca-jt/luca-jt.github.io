Zlider: A Simple Slide Show Program
I always resented the way most slide show programs work - a pay wall, a file format with seemingly non-deterministic formatting output or poor open source alternatives. This slide show program is all I need for almost all use cases and uses text files and a simple "markup language" for the slide show files.
